# DevToys.JsonToGo

[![NuGet](<https://img.shields.io/nuget/v/DevToys.JsonToGo>)](<https://www.nuget.org/packages/DevToys.JsonToGo/>)

The JSON to Go struct conversion tool for DevToys.

## Installation

1. Download the `DevToys.JsonToGo` package from NuGet.org.
2. In DevToys, open `Manage Extensions`, click `Install an extension`, and select the downloaded package.
